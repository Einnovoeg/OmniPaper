# Board Targets

## M5Paper
- PlatformIO env: `m5paper_release`
- MCU: ESP32
- Firmware binary: `omnipaper-<tag>-m5paper-firmware.bin`

## M5PaperS3
- PlatformIO env: `m5papers3_release` for release bundles, `m5papers3` for normal development builds
- MCU: ESP32-S3
- Firmware binary: `omnipaper-<tag>-m5papers3-firmware.bin`

## LilyGo T5 4.7 Plus S3
- PlatformIO env: `lilygo_epd47`
- MCU: ESP32-S3
- Firmware binary: custom build from `pio run -e lilygo_epd47`
